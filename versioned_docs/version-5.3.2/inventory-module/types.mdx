## Enums

### Equipment Slot Action
`Equipment Slot Action` enum value describes type of the action done to
an equipment slot.
Values:
* `Set`
* `Cleared`

### Item Update Type
`Item Update Type` enum value describes type of the update done to an item.
Values:
* `Picked Up`
* `Updated`
* `Used`
* `Equipped`
* `Unequipped`
* `Dropped`
* `Before Destroy`
